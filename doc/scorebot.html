ScoreBot Type

The ScoreBot object, like the game data structures, is a Python type residing in the fragmon module. The ScoreBot type is a base class that allows the user to easily extend the functionality of Fragmon's crude scorebot. Simply subclass ScoreBot and implement some or all of its methods. See the gamequery/query_scorebot.py module for an example implementation.



Members

These members are inherited by custom scorebots. Note that most mutable variables are updated before each broadcast; format is updated with the current format from the user's configuration, and the competition data (title, name*, score*, time) is copied from the user-editable competition section of the Fragmon window. Changes to most variables should be made in the broadcast() method before calling sub_format().

Methods

Methods inherited by custom scorebots. Methods tagged Virtual can be overridden by Python code. See query_scorebot.py for an example implementing many of these methods.

Name

Return Value

Description

broadcast()

None

(Virtual) Override this method to change the broadcasting behavior. The default behavior is to call sub_format() and dump the output to the channel configured by the user.

broadcast_players()

None

(Virtual) Called when the Fragmon user presses CTRL-L or clicks Print Players. Override this method to change the broadcasting behavior. The default behavior is to simply print the players on each team, one line per team.

__cleanup__()

None

(Virtual) Called by Fragmon before destroying the bot's base object. See query_scorebot.py for an example implementation.

get_overview()

Overview object

Gets a copy of the Overview object from the last query made.

get_players()

List of Player objects

Gets a copy of the list of Player objects from the last query made.

get_rules()

List of Rule objects

Gets a copy of the list of Rule objects from the last query made.

sub_format(fmt, vars)

string

Method takes an optional format string as the first argument and a list of variables to add to the basic format specifiers. Both arguments are optional. By default, it uses the format member if the fmt argument is not supplied. Returns a string with all variables substituted. See query_scorebot.py for example usage.

get_option(key)

string

(Virtual) get value for key. See query_scorebot.py for an example.

set_option(key, value)

string or None

(Virtual) set value for key, optionally returning a string error message if the value is invalid. See query_scorebot.py for an example.
