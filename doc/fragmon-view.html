Fragmon Main Window Sections



Server Input and Query Info

This section allows the user to pick servers and shows an overview of basic query data

Competition Info

Representing additional data from the Overview, fields from left to right: team A name, team B name, team A score, team B score, and time. Fields in this section are user editable for broadcasting with a scorebot. It may be useful to change the generic team names to clan names, or input the scores manually. If you type text manually into any of these fields, the text will remain in place until it is removed. When it is removed, the Fragmon-generated value will return (such as Red or Blue).

Players

List of players on the server. Teams can be overridden by highlighting players in the list and selecting the team in the right click menu or by typing A, B or F. Team overrides can be cleared by selecting the option in the right click menu.

Rules

List of rules on the server.
