FRAGMON PINGTOOL



Fragmon is a FPS pingtool and scorebot for Windows and Unix. Take a look at the screenshots above to see what it looks like in action. Check out the docs here.



DESIGN GOALS

  • Portability works on Windows, Linux, and possibly other Oses

  • Pingtool includes simple game server query support with a portable Python library

  • Score Broadcast broadcast scores on IRC easily with Fragmon with its builtin scorebot

  • Extensibility support new query protocols and create custom scorebots easily by adding simple Python modules



DOWNLOAD

Current version of Fragmon is 0.2, with Windows installer and Linux binaries available below. Source code is also available under the MIT license, so download the source if you wish. You'll need Python 2.4 and wxWidgets 2.6 installed. The Linux binaries require at least GTK 2.0

Binaries:

  • Windows Installer (Python libs included)

  • Windows Installer (Requires full Python installation)

  • Linux (Debian, statically linked to Python 2.4 – Python libs included)

  • Linux (Debian, dynamically linked to Python 2.4 – requires full Python installation)

  • Linux (Debian, dynamically linked to Python 2.4 and wxGTK 2.6.2 – requires Python and wxGTK installation)
